Achievements

Human Pose Estimation Approach Yoga Pose Estimation

Method for automatically detecting yoga poses in realtime or the recorded videos. Used Mediapipe to extract body landmarks. And to make the prediction used the Multilayer Perceptron (MLP) classifier and Long ShortTerm Memory (LSTM) for training the model.

NU Equipment Management System

A web application with two types of users one acting as an admin and the other as the authorities in charge of the equipment was created using PHP and Mysql. CRUD operations are used to maintain the equipments data and maintainence requirements. And report can be generated.

Stock Sentiment Prediction Analysis Using News Headlines

Collected a dataset from kaggle.com. Extracted features from news headlines and then used Random Forest Classifier Multinomial Naive Bayes and Passive Agressive Classifiers for analysis. Analyzed sentiments in 2 class one for increasing stock price and other for decrease in stock price.

Word Cloud Generator

Built a web application based on django framework. Numpy is used for converting the image into array. The word cloud generator is easy to use, the user just need to put the text and have options like removing stopwords, word cloud shape etc and then it will generate the wordcloud
